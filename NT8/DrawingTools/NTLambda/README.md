# NTLambda NT8 Drawing Tools

Native drawing tools included in the clean NT8 pack:

- `NTLAnchoredVwap` - anchored VWAP drawing tool with deviation bands.
- `NTLFixedRangeVolumeProfile` - fixed-range volume profile with start/end anchors, plus optional open-ended mode from start to latest bar.

Profile-style range analysis is also provided by the `NTLRangeVolumeProfile` indicator in `Indicators/NTLambda`.
