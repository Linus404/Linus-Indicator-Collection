# NTLambda NT8 Chart Styles

Native chart styles included in the clean NT8 pack:

- `NTLSimpleHighLowChart` - simple high/low chart style with configurable line colors and width.
- `NTLFootprintChartStyle` - footprint-style chart style.
- `NTLTpoChartStyle` - filled-square TPO/market-profile chart style.
- `NTLTpoLettersChartStyle` - outline-only TPO chart style with letters.
